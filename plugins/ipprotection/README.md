# Firefox IP Protection Plugin

Skills and tools for Firefox IP Protection (VPN) development.

## Skills

### `ipprotection-review`

Review guidance for Firefox's built-in IP Protection (VPN) module, covering panel UI, proxy manager, telemetry, Fluent strings, and enterprise policy.

Use this skill when reviewing changes under `browser/components/ipprotection/` or `toolkit/components/ipprotection/`. See [`skills/ipprotection-review/SKILL.md`](skills/ipprotection-review/SKILL.md) for details.
